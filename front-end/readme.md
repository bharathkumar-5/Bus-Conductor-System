## Frontend – Booking & Boarding Interface

### What I Built

I developed a **clean, responsive, and conductor-friendly UI** that supports fast booking and boarding operations.

---

### Key Implementations

* Booking form with create and edit modes
* Mobile number–based booking detection
* Interactive seat selection with real-time validation
* Clear seat state visualization
* Booking confirmation popup
* Boarding list with quick actions
* Click-to-call mobile integration

---

### UX Decisions

* Minimized clicks for common conductor workflows
* Used visual cues instead of text-heavy messages
* Prevented errors by disabling invalid actions
* Ensured responsive behavior across screen sizes

---

### Tech Stack

* React 18
* TypeScript
* Vite
* Tailwind CSS
* Axios
* Lucide Icons

---

### Running the Frontend

```bash
npm install
npm run dev
```

---

### Environment Variables

```env
VITE_API_URL=http://localhost:5000/api
```
